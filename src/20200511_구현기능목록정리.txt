회원관리

	- 회원가입
	- 로그인
	- 로그아웃
	- 비밀번호 수정
	- 포인트 추가/사용
	
	= 회원 탈퇴
	= 포인트 로직
	= 회원등급 책정 로직
	
회원별 결제내역 관리

	- 결제내역이 가진 자료형 기초상태 구현함.
	
	= 상세한 내역(년/월/일) 미구현 상태.  
	

상품목록+매장재고관리

	- 카테고리별 상세 출력 기능 구현 완료
	- 최소필요수량 도달 시 부족알림 뜸.
	- 발주에서 재고가 가진 고유한 배열에 데이터 입력 확인 완료
	
발주관리
	
	- 거래처가 가지는 정보저장기능 구현 완료.
	- 거래처 추가, 수정, 삭제 구현 완료.(수정 로직에 대한 검증 필요함)
	- 거래처에서 물품발주기능 구현 완료.
	- 거래처에서 발주 시 발주기록 구현 완료.
	
	= 발주 년,월,일 단위 결제내역 상세구현 미완.
