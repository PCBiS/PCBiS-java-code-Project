회원별 결제내역 관리

	= 상세한 내역(년/월/일) 미구현 상태.  
	

상품목록+매장재고관리

	- 카테고리별 상세 출력 기능 구현 완료
	- 최소필요수량 도달 시 부족알림 뜸.
	- 발주에서 재고가 가진 고유한 배열에 데이터 입력 확인 완료
	
	= 판매관리가 구현되지 않아서 재고에서 제품이 빠질 때에 대한 검증이 없음.
	
발주관리
	
	- 거래처가 가지는 정보저장기능 구현 완료.
	- 거래처 추가, 수정, 삭제 구현 완료.(수정 로직에 대한 검증 필요함)
	- 거래처에서 물품발주기능 구현 완료.
	- 거래처에서 발주 시 발주기록 구현 완료.
	
	= 발주 년,월,일 단위 결제내역 상세구현 미완.
	= 거래처가 가지는 아이템 목록을 독립 된 Product(제품) 단위로 독립 해볼것을 주문(05월 11일)하셨음. - 로직 수정할 부분이 많아서 현실적으로는 더이상은 무리.
